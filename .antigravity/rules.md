# 앱 단계별 상세 UI 및 기능 구현 규칙

## 1. 사진 엔진 (Photo Engine)
- **UI 교체**: 기존의 '글씨체 변경', '기본/감성' 버튼은 삭제한다.
- **프로세스**: 
  - `<더 멋있게 만들기>` 버튼 클릭 시 **AI 에이전트 3개(회화 스타일, 선명도 개선, 콜라주 생성)**를 선택할 수 있는 화면 노출.
  - `<요구사항을 적어 주세요>` 입력창 제공.
- **내비게이션**: 결과물 확인 후 하단에 `<이전>`, `<이 사진으로 할래요>`, `<다음 글쓰기>` 버튼 배치.

## 2. 글쓰기 엔진 (Writing Engine)
- **이원화 UI**:
  - `<내가 쓸게요>`: 폰트, 크기, 색상 조절이 가능한 에디터 노출.
  - `<AI에게 부탁하기>`: 에이전트가 사진 분위기를 읽고 글귀 자동 생성.
- **내비게이션**: 완료 단계에서 `<됐어요>`, `<사진과 글이 잘 어울려요>`, `<다음 음악 만들기>` 버튼 순차 배치.

## 3. 음악 엔진 (Music Engine)
- **모드**: `<선택>` (BGM 목록 제시) / `<AI에 부탁>` (Generative Music 생성).
- **내비게이션**: `<이전>`, `<이 음악으로 할래요>`, `<다음 목소리>` 버튼 배치.

## 4. 목소리 엔진 (Voice Engine)
- **기능**: `<녹음>`, `<듣기>` (반복 청취 지원).
- **AI 변환**: `<다른 목소리로>` 버튼 클릭 시 AI 에이전트가 목소리 톤을 변환 (시 낭송, 웅장한 톤).
- **저장 시스템**: PC 내 '나의 인생 갤러리' 폴더를 자동 생성하여 결과물을 일괄 저장.
- **내비게이션**: `<이전>`, `<발표 준비>` 버튼 배치.

## 5. 발표 및 무대 매니저 (Stage Manager)
- **공유 기능**: SNS 목록을 카드 형태로 제시, 선택 시 업로드.
- **무대 기능 (초대)**: Google Contacts MCP 연동으로 연락처에서 초대 명단 정리.
- **4대 체크리스트**: [발표일정], [무대준비], [사람/업체], [의상/화장] 체크리스트 제공.
